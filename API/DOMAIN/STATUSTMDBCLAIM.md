# StatusTMDBClaim

## DESCRIPTION
Queries properties related to an existing claim notice from the trademark database.

## AVAILABILITY
All users have access to this command.

----
## COMMAND

Parameter | Min | Definition | Type
---- | ---- | ---- | ----
COMMAND | 1 | `StatusTMDBClaim` | COMMAND
CLAIMKEY | 1 | The key of the claim notice | TEXT or NULL

----
## RESPONSE

Code | Description
---- | ----
200 | Command completed successfully
420 | Command failed due to server error. Server closing connection
421 | Command failed due to server error. Client should try again
423 | Command failed due to server error. Client should try again (Could not get session)
500	| Invalid command name
503 | Invalid attribute name
505 | Invalid attribute value syntax
530 | Authentication failed

Property | Min | Max | Definition | Type
---- | ---- | ---- | ---- | ----
NOTICE-ACCEPT-ID | 1 | 1 | The claim notice's accept ID | TEXT
NOTICE-CREATEDDATE | 1 | 1 | Creation date of the claim notice | DATETIME
NOTICE-EXPIRATIONDATE | 1 | 1 | Expiration date of the claim notice | DATETIME
NOTICE-POA-ACCEPT-HASH | 1 | 1 | A hash of the claim notice which needs to be stated when registering a domain for which a claim exists. | TEXT
NOTICE-TEXT | 1 | 1 | The claim notice's text | TEXT
NOTICE-XML | 1 | 1 | The whole answer from the trademark database in XML format | TEXT

----

```
[COMMAND]
COMMAND = StatusTMDBClaim
CLAIMKEY = 2016121201/C/9/F/yfVfSITVpAjOKZu0FnjjLbc-0000000263
EOF
```
